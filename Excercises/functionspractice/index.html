Functions Practice Excercise

In this exercise, you will practice writing and using functions in JavaScript

Task 1: Create a Function to Greet a User

Create a function named greetUser that takes a single parameter, name, and returns a greeting message in the format: "Hello, [name]!"

Task 2: Create a Function to Calculate the Area of a Rectangle

Create a function named calculateArea that takes two parameters, length and width, and returns the area of the rectangle.

Task 3: Create a Function to Check if a Number is Even or Odd

Create a function named isEven that takes a single parameter, number, and returns true if the number is even and false if it is odd.

Task 4: Create a Function to Find the Maximum of Three Numbers

Create a function named findMax that takes three parameters, a, b, and c, and returns the maximum of the three numbers.
